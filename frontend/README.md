# Escrow Frontend

React frontend for the Solana Escrow dApp.

## Live Demo

https://frontend-wheat-omega-64.vercel.app/

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- @solana/wallet-adapter-react
- @coral-xyz/anchor

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Connect Phantom/Solflare wallet
- Create escrows (lock SOL for recipient)
- View sent/received escrows
- Accept escrows (as recipient)
- Cancel escrows (as sender)
